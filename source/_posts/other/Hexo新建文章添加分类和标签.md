#### 教程

干货教程，不多废话，使用 [Typora](https://www.typora.io/) 打开编辑：添加tags和categories

<!-- more -->

```
title: Hexo新建文章添加分类和标签  #文章标题
date:  2018-06-28 10:15:32 #文章日期
categories: extends-end  #文章目录分类
tags: #文章标签
- Hexo
- 网站
```

这样写完之后，然后 部署，发布，就可以看到了，本地可以先使用 Hexo server 命令本地预览

```
Hexo server
```
